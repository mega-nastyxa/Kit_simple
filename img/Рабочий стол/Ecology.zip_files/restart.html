Sheet 11: slow_roads_lines

Проблемность дорог (по линиям) slow_roads_lines.geojson
Описание слоя Отношение средней скорости движения к разрешенной по участкам дорожной сети. Замедления - события, собираемые по открытым данным Яндекс-навигаторов (замедления-пробки исключены автоматическим образом). События привязаны к участкам дорог графа для визуального представления и последующего анализа. Территория , в основном, между населенными пунктами
Источник Геоинтеллект, Yandex
Тип линейный
Список полей:
id Уникальный идентификатор
category Числовой идентификатор категории дороги
category_name Числовой и именной идентификатор категории дороги
dates Дата актуальности
free_speed_to_limit Отношение средней скорости к разрешенной на участке
length Длина, м
region Регион

Sheet 12: auto_foot_graph_arh_obl

Граф дорог auto_foot_graph_arh_obl
Описания слоя Граф дорог по Архангельской области. Участки дорог от перекрестка до перекрестка с атрибутами максимальной скорости по знакам
Источник OSM, Геоинтеллект
Тип линейный
Список полей
fclass Класс дороги по OSM
name Название
ref Код
oneway Одностороннее движение (F - вперед, T - назад, B - в обе стороны)
maxspeed Максимальная скорость по знакам на участке дороги, км.
layer Уровень участка относительно других участков (вертикальные отношения)
bridge Мост (Y -да, N - нет)
tunnel Тоннель (Y -да, N - нет)
auto Можно ли проехать на машине (Y -да, N - нет)
foot Можно ли пройти пешком (Y -да, N - нет)
meters Протяженность в метрах

minutes_foot Минуты на прохождение участка пешком со скоростью 4.5 км/ч
